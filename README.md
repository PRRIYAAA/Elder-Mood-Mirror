## üí° **About the Project**

Elder Mood Mirror (EMMA) is an AI-powered web platform created to enhance the emotional and mental well-being of senior citizens. It combines real-time mood detection through camera-based facial emotion analysis with daily wellness surveys to help elders track their emotional health and receive timely support.

As people age, emotional challenges like loneliness, anxiety, and mild depression often go unnoticed, especially when families live apart or elders have limited social interaction. While family care remains central to emotional well-being, it can be difficult for loved ones to consistently monitor subtle mood changes amidst daily responsibilities.

EMMA is designed not to replace family support, but to strengthen it. By offering continuous emotional monitoring, automatic alerts, and communication channels between doctors and guardians, EMMA ensures that families stay informed, connected, and responsive to their elder‚Äôs mental state.

Through AI-driven insights and shared visibility across caregivers, doctors, and family members, EMMA empowers a collaborative ecosystem of care  promoting early intervention, emotional stability, and a sense of belonging for elders
---

## üåü **Key Features**

1. **Dual Login System:** Separate logins for elders and doctors with role-based dashboards.
2. **AI Mood Detection:** Real-time emotion recognition through the elder‚Äôs webcam.
3. **Mood & Wellness Surveys:** Quick daily surveys to record emotional states.
4. **Mood-Based Support:** Personalized recommendations for improving wellness.
5. **Doctor Dashboard:** Displays total patients, active alerts, appointments, and messages.
6. **Patient Profile View:** Shows each elder‚Äôs medical condition, disability info, and emotional trends.
7. **Guardian Communication:** Enables doctors to chat directly with guardians for coordinated care.
8. **Alert Notifications:** Automatically flags elders who need immediate attention.
9. **Wellness Reports:** Graphical view of mood and emotional progress.
10. **Data Security:** End-to-end encrypted communication ensuring privacy.

## ‚öôÔ∏è **Installation & Setup**

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/Elder-Mood-Mirror.git
   cd Elder-Mood-Mirror
   ```
2. Install dependencies:

   ```bash
   npm install    # or pip install -r requirements.txt
   ```
3. Configure environment variables in `.env` (database URL, API keys, etc.)
4. Run the development server:

   ```bash
   npm start      # or python app.py
   ```
5. Open `http://localhost:3000` in your browser.
